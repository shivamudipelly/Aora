## ğŸš€ Getting Started

### Prerequisites

- Node.js 16+
- Expo CLI
- iOS Simulator or Android Emulator (or physical device)

### Installation

```bash
# Clone repository
git clone https://github.com/shivamudipelly/Aora.git
cd Aora

# Install dependencies
```
